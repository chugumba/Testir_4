Этот код решает следующую задачу на языке C++^
Даны координаты 3-х вершин на плоскости. Вычислить периметр и площадь части треугольника, лежащей в 3-м квадранте (если он существует).

Вначале вводятся координаты 3-ёх точек, после этого код определяет точки пересечения сторон исходного треугольника с осями координат в 3 квадранте. После этого, точки упорядочиваются
от наименьшей по X (если координаты 2 точек по X совпадают, то выбирается наибольшая по Y), потом идут от наименьшей по Y к наибольшей по Y, потом от наибольшей по X к наименьшей по Y.
Определяется колличество точек в 3 квадранте и на основе этого происходит вычисление периметра и площади фигуры с помощью теоремы Пифагора и формулы Герона. 
Из-за используемого способа нахождения площади и периметра упорядочить точки - необходимо, иначе результат вычислений будет некорректен.


Примеры значений для проверки:
1 4
-3 -3
5 -3

-3 -3
1 -3
-1 1

-7 2
-7 -4
-5 -4

-1 -5
0 -4
1 -5

0 1
-1 -2
1 -2

-1 -1
1 -1
0 1

-1 -1
1 -1
0 1
